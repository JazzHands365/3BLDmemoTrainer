# 3BLDmemoTrainer
A tool for learning to memorize letter sequences for 3BLD.\
I decided to make these as html files as I wanted to be able to use these at school, where the laptops are locked down and unable to run most common application formats.\
This is a very basic tool that allows for you to develop your words for letterpairs. It also serves as a way to practice memoing without having to actually hava a cube with you. You generate a new string of letters and memorize them in whatever way you please. You then hide the letters and review your images or words. Next you play the game to simulate distractions or other stresses on your brain while executing. When finished, you input your memorized sequence into the answer bar.\
memo-trainer.html is a barebones version of the program, while memo-trainer (with game).html is a bit more fleshed out.
